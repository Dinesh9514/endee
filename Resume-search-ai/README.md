# AI Resume Search

## Overview
This project demonstrates a simple AI resume search system using vector embeddings.

## Features
- Converts text into embeddings
- Stores vectors in FAISS
- Searches similar resumes

## Technologies
- Python
- Sentence Transformers
- FAISS

## How to Run

Install dependencies

pip install -r requirements.txt

Run project

python app.py

## Vector Search Concept

This project converts resumes into embeddings and stores them in a FAISS vector database. When a query is given, the system finds the most similar resumes using vector similarity search.
